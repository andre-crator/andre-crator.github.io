# Checklist de Entrega – Landing Page

## Conteúdo
- [ ] Nome do negócio
- [ ] Texto do hero (título + subtítulo)
- [ ] Serviços (3 blocos)
- [ ] Links (WhatsApp / Instagram / Google Maps / etc.)
- [ ] Logo (se tiver)
- [ ] Imagens (3 a 6 imagens)

## Página
- [ ] CTA principal aponta para WhatsApp
- [ ] Seção “Serviços” clara
- [ ] Seção “Projetos” ou “Portfólio”
- [ ] Contato completo
- [ ] Responsivo (celular)

## Publicação
- [ ] Repositório criado
- [ ] GitHub Pages ativado
- [ ] Link publicado testado
- [ ] Print/preview gerado
- [ ] README preenchido

## Pós
- [ ] Registrar no portfólio (link + 2 linhas de contexto)
- [ ] Post no LinkedIn mostrando evolução/entrega

# Processo Replicável (Copiar → Trocar → Publicar)

## Etapa 1 — Clonar/duplicar template
- Copie `template/landing-page`
- Renomeie para o cliente:
  - `cliente-nome/`

## Etapa 2 — Conteúdo
- Edite `conteudo.md` primeiro
- Depois ajuste `index.html` para refletir (se necessário)

## Etapa 3 — Branding
- Troque logo e imagens em `assets/`
- Ajuste cores em `style.css` (variáveis no :root)

## Etapa 4 — Publicação
- Suba para GitHub
- Ative GitHub Pages (branch main / root)
- Teste no celular

## Etapa 5 — Registro no Portfólio
- Adicione o link no seu site/portfólio
- Escreva 2 linhas sobre:
  - problema do cliente
  - solução entregue
  - resultado esperado

## Padrão de commit (recomendado)
- feat(template): ...
- docs(template): ...
- refactor(template): ...

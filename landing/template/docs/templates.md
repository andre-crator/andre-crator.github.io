# Templates (Produtos Reutilizáveis)

Este repositório contém templates prontos para replicação rápida.

## 1) Landing Page – Pequenos Negócios
**Objetivo:** criar páginas profissionais com foco em conversão e contato rápido.

- Pasta: `template/landing-page`
- Ideal para: serviços locais, autônomos, negócios pequenos
- Canal principal: WhatsApp / Instagram

### Como usar
1) Copiar pasta
2) Trocar `conteudo.md`
3) Ajustar imagens/cores
4) Publicar
